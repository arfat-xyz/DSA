Accessing Methods in JavaScript

What is Accessing?
Answer: Accessing means getting the value of a particular element at a specific index.

Methods for Accessing:
Answer: Direct Indexing (Array): const data = [10, 20, 30]; console.log(data[1]); // Outputs 20,


at(): const data = [10, 20, 30]; console.log(data.at(-1)); // Outputs 30 (last element),


find(): const data = [10, 20, 30]; console.log(data.find(item => item > 15)); // Outputs 20,


findIndex(): const data = [10, 20, 30]; console.log(data.findIndex(item => item > 15)); // Outputs 1 (index of 20),


filter(): const data = [10, 20, 30]; console.log(data.filter(item => item > 15)); // Outputs [20, 30],


map(): const data = [10, 20, 30]; console.log(data.map(item => item * 2)); // Outputs [20, 40, 60],


Destructuring Assignment (Array): const data = [10, 20, 30]; const [first, second] = data; console.log(first); // Outputs 10,


Dot Notation (Object): const person = { name: "John", age: 30 }; console.log(person.name); // Outputs "John",


Bracket Notation (Object): const person = { name: "John", age: 30 }; console.log(person["age"]); // Outputs 30,


Object.keys(): const person = { name: "John", age: 30 }; console.log(Object.keys(person)); // Outputs ["name", "age"],


Object.values(): const person = { name: "John", age: 30 }; console.log(Object.values(person)); // Outputs ["John", 30],


Object.entries(): const person = { name: "John", age: 30 }; console.log(Object.entries(person)); // Outputs [["name", "John"], ["age", 30]],


hasOwnProperty(): const person = { name: "John", age: 30 }; console.log(person.hasOwnProperty("name")); // Outputs true,


Dot Notation for Nested Objects: const person = { name: "John", address: { city: "New York" } }; console.log(person.address.city); // Outputs "New York",


Bracket Notation for Nested Objects: const person = { name: "John", address: { "home city": "New York" } }; console.log(person["address"]["home city"]); // Outputs "New York",


Destructuring Nested Objects: const person = { name: "John", address: { city: "New York" } }; const { address: { city } } = person; console.log(city); // Outputs "New York",


Accessing Nested Arrays: const data = [[1, 2], [3, 4], [5, 6]]; console.log(data[1][0]); // Outputs 3,


String.charAt(): const str = "Hello"; console.log(str.charAt(1)); // Outputs "e",


String.at(): const str = "Hello"; console.log(str.at(-1)); // Outputs "o" (last character),


Array.slice(): const data = [10, 20, 30]; console.log(data.slice(1, 3)); // Outputs [20, 30],


Array.splice(): const data = [10, 20, 30]; data.splice(1, 1, 25); console.log(data); // Outputs [10, 25, 30].
